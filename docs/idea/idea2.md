# 同项目运行多个模块

(这里以Maven项目为例)

*************

## 创建多个模块

1. 按照正常步骤创建项目
2. 这个时候 src 资源目录是父项目的，建议删除
3. pom 文件资源为子项目共享的
4. 右键父目录添加 module
![img.png](img.png)
5. pom 只在当前模块及其子模块中生效
6. 但项目多模块设置完毕

*********
